Want to Organize a Brigade?

Thats fantastic! We are so excited you and your team are interested in joining this awesome network of developers, community organizers, story-tellers, government partners, and caring residents!

Ready to Get Started?

Code for America Brigades organize through the 4Ps: people, plans, partnerships, and participation. A group can join the Code for America Brigade network as a Brigade Network Member or an Official Brigade Chapter.

    Brigade Network Members

    A Brigade Network Member is a local Brigade committed to organizing people and participating in the Code for America Brigade network.

    We ask groups demonstrate this by:

    1. People

    Hosting consistent hack nights (at least three)

    Having a leadership team

    Sharing a Code of Conduct on your website

    2. Participate

    Share ideas on the Code for America Slack channel

    Contibute content to the CFAPI

    Once your ready to join, sign up for the next Brigade Network Member onboarding session FILL OUT THIS FORM. We will host these calls once a month beginning January 2015.

    Once you’ve been onboarded, we’ll invite your leadership team to participate in the Code for America Brigade Slack channel and the CFAPI.



    Official Brigade Chapters

    An Official Brigade Chapter is local Brigade committed to organizing people, participating in the Code for America Brigade network, and working with municipal and community partners to execute on a strategic plan that aligns with the Code for America Principles for 21st Century Government.

    We ask groups demonstrate this by:

    1. People

    Hosting consistent hack nights (at least three)

    Having a leadership team

    Sharing a Code of Conduct on your website

    2. Plans

    Create a plan that aligns with the Code for America Digital Government Principles.

    3. Partners

    Collaborate within local government and local community groups and nonprofits.

    4. Participate

    Share ideas on the Code for America Slack channel

    Contibute content to the CFAPI

    Once your ready to join, sign up for the next Official Brigade Chapter onboarding session FILL OUT THIS FORM. We will host these calls once a month beginning January 2015.

    Once you’ve been onboarded, we’ll invite your leadership team to participate in the Code for America Brigade Slack channel and the CFAPI.
